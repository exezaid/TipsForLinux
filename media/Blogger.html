instalar o Grub na mbr

Monday, July 26, 2010 | Posted by Gustavo | labels: mbr

Para instalar o Grub na mbr, use o comando grub-install /dev/?dispositivo? exemplo: grub-install /dev/hda coloca o grub na mbr do primeiro hd ide (master da primária) Em seguida rode o comando grub (isso vale também se você deu boot num disco com grub !). Ao rodar o comando grub aparece um prompt grub. grub> root (hd0,0) # o /boot estava /dev/hda1 grub> install /grub/stage1 d (hd0) /grub/stage2 p /grub/menu.lst quando não boota e entra no grub grub> root (hdX,Y) grub> kernel = /vmlinuz root=/dev/hda3 grub> initrd = /initrd Para criar um disco de boot do Linux bem rapido, use os comandos abaixo: # dd if=/path/do/seu/kernel of=/dev/fd0 # rdev /dev/fd0 /dev/hdXY Onde X é o seu HD(hda, hdb, sda, sdb, etc) e Y é a partição em que se encontra o diretório / (raiz)

Dicas úteis para usar o tar ou relacionado a ele.

Monday, July 26, 2010 | Posted by Gustavo | labels: mbr

juntar e zipar
$ tar -czvf arquivo.tgz *.txt
para abrir
$ tar -xzvf arquivo.tar.gz
$ tar -xjvf arquivo.tar.bz2
apagando vários arquivos !! Se abrir um tgz no lugar errado.
$ tar tfz nomefile.tar.gz | xargs rm
ou se possuir um arquivo com os nomes dos arquivos a serem apagados.
$ cat arquivos| xargs rm
ou ainda ...
$ rm `find . -name \*.tmp`
Compactar um diretório e dividí-lo em vários disquetes.
$ tar czvM -f /dev/fd0H1440 diretório
usar o tar excluindo um arquivo
$ tar --exclude /root/sis1/sis1.tgz -czvf sis1.tgz /
usar o tar só em uma partição local como barra /
$ tar -zcvlf barra.tgz /
Compactar e enviar via ssh
$ tar -czf - diretorio/ |ssh usuario@servidor 'cat >arquivo.tgz'
